Python_配置Python编程环境

2020-05-07 / 1 min read

提示：基于Python3的基础语法

1.搭建Python编程环境

  1. 安装Python
  • [下载链接] (http://python.org/downloads/)
  • 打开页面后将会看到下载选项，单击下载Python3。
  • 选中复选框Add Python to PATH，配置Python环境变量。
  • 打开命令窗口，输入python，如果出现Python提示符就说明Python已经安装成功。👌
  1. 配置编辑器
    我使用的是Visual Studio Code编辑器，下载链接在这里👉链接
    打开后找到扩展搜索Python下载即可。
