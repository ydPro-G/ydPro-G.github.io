Python_变量和简单数据类型

2020-05-08 / 2 min read

1.认识变量

  1. 什么是变量？
     message = "Hello World!"
print(message)
    message就是一个变量，每个变量都能储存一个值。值——与变量相关联的信息。
    提醒：在程序中可随时修改变量的值，而Python会始终记录最新值🙌
  1. 变量命名规则
    • 变量名只能包含字母，数字，下划线。
    • 不能用数字开头。
    • 变量名不能包含空格。
    • 不要用Python关键字和函数名做变量名
    • 变量名要简单而又具有描述性，易懂。😀

2.字符串

  1. 什么是字符串？

字符串就是一系列的字符。如下所示👇
“This is a string."
'This is also string.'
2. 修改字符串的大小写

name = "angel"
print(name.title())
  • 方法：title() 字符串首字母大写
  • 方法：upper() 字符串全部大写
  • 方法：lower() 字符串全部小写
  1. 合并字符串
first_string = "A"
last_string = "B"
full_string = first_string + last_string + "."
print(full_string)
  • 使用+号来合并字符串
  1. 添加换行符或制表符
print("\nPython") 
print("\tPython")
  • \n:换行符
  • \t：制表符
  1. 删除空白
favorite_language = ' python '
 favorite_language.rstrip()
' python'
 favorite_language.lstrip()
'python '
 favorite_language.strip()
'python'
  • .rstrip()：删除末尾空白
  • lstrip()：删除开头空白
  • strip()：删除两端空白

3.数字

  1. 整数：
  • 2 + 3
  • 3 - 2
  • 2 * 3
  • 3 / 2
  • 2 ** 3：使用两个乘号表示乘方运算
  1. 浮点数
  • 0.1+ 0.1
  • 0.2 + 0.2
  • 2 * 0.1
  • 2 * 0.2
  • 其结果包含的小数位数可能是不确定的。
  1. 使用函数str()避免类型错误
age = 18
message = "Happy " + str(age) + "rd Birthday!"
print(message)
  • 使用函数str()将值为整数的变量用作字符串(将非字符串值表示为字符串)🤙
  1. 注释
# 打招呼
print("Hello!")
  • 注释使用#号表示，井号后面的内容会被Python解释器忽略
  • 编写注释主要目的是阐述代码要做什么，如何做
